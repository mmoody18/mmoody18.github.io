Website Evaluations

Apple

Apple’s website is a prime example of a well-executed design that emphasizes clean visuals and intuitive navigation.

  • Contrast: The site uses high contrast between text and background, ensuring that information is easy to read.
  • Repetition: Consistent typography, color schemes, and layout patterns create a harmonious and professional look.
  • Alignment: All elements are thoughtfully aligned, lending the page a clear structure and organized appearance.
  • Proximity: Related content is grouped together, which helps users quickly identify and understand key information.

Xray Products - pnwx.com

pnwx.com is an outdated website that struggles with basic design principles, making it hard for users to navigate and read its content.

  • Contrast: The text often fades into a similarly hued background, severely reducing readability.
  • Repetition: Inconsistent use of fonts and colors creates a disjointed visual experience.
  • Alignment: Elements are misaligned and haphazardly arranged, contributing to a confusing layout.
  • Proximity: Overcrowded content with little spacing overwhelms visitors and obscures key information.
